# 📊 Student Performance Analysis & Visualization

## 📌 Overview
This project presents a visual analysis of student academic performance using semester-wise SGPA and CGPA data. The objective is to identify trends in improvement, highlight top-performing students, and understand overall class performance.

## 🛠️ Tools Used
- Microsoft Excel  
- Data Cleaning & Processing  
- Data Visualization  

## 📂 Project link
- [reports/ → PDF containing all visualizations and analysis ](https://public.tableau.com/app/profile/tadepalli.haneesh/viz/StudentsSemesterResultAnalysisReport/Dashboard1) 

## 📊 Key Insights
- Clear upward trend in SGPA for many students, indicating academic improvement  
- Top-performing students maintain consistently high CGPA  
- Strong correlation observed between SGPA improvement and final CGPA  
- Performance gap exists between top-tier and average students  

## 📈 Visualizations Included
- SGPA progression analysis  
- CGPA distribution  
- Top 10 students by CGPA  
- CGPA vs Improvement comparison  
- Class performance summary
